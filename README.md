# Mapping Nepal

*Mapping Nepal* is a Shiny web application designed to help researchers, students, and GIS users easily visualize and extract maps of Nepal at multiple administrative levels. The app allows users to customize map appearance, add labels, and download high-quality map outputs suitable for journal publications and reports.

# Features

## Multi-level Mapping

-   Generate maps at **national**, **provincial**, **district**, and **municipality** levels.

## Customization Options

-   Apply different **map themes**.
-   Toggle **labels** (province/district/municipality names).
-   Customize map appearance for **publication-ready** outputs.

## High-Quality Downloads

-   Export maps in **PNG** and **PDF** formats with high resolution.
